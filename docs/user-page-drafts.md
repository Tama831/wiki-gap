## 参考情報

### Babel テンプレート

`{{Babel|ja|en-2}}` で「日本語ネイティブ + 英語中級」を表示。
レベルは:
- `ja` (ネイティブ) / `ja-N` (ja-N も同じ意味、Native level)
- `en-1` (初級) / `en-2` (中級) / `en-3` (上級) / `en-4` (ネイティブ並)

たまさんが英語論文を普通に読める前提なら **`en-3`** が実際に近いかも。

### 投稿の手順

1. https://ja.wikipedia.org/wiki/利用者:Goyama2128 を開く
2. 「ページを作成」(まだ存在しないので) → 上のいずれかの wikitext を貼り付け
3. 編集要約: `利用者ページ初版`
4. 「変更を公開」で完成

### 追加で書ける項目 (任意・お好みで)

- 趣味 (パン・ピザ作り / 多肉植物 / アニメ / ゲーム音楽 / DIY) → 親近感
- 「現在翻訳中の記事」セクション → 進行中作業の見える化 (Wikipedia 編集者によくある慣行)
- COI 開示 ([[Wikipedia:利益相反]]) → 製薬企業との関係などがある場合の自主開示 (該当なければ書かない)
- {{User Translator|en|ja}} などの「翻訳者」テンプレート

### 書かない方が良い項目

- 健康状態 / プライベートな事情
- 具体的な所属医療機関名
- 連絡先メールアドレス (トークページが原則)
- 本名 (Goyama2128 がアカウント名なのでそれで一貫)
